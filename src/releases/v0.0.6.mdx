## UI Improvements

### New Components
- `Avatar` - User avatars with fallback
- `Dropdown` - Menus with click-outside-to-close
- `Card`, `Badge`, `Divider`, `Modal`
- `Input`, `Textarea`, `Select`, `Range`
- `Button` with variants and tooltips
- `Heading` components (H1-H6)
- `Loader` and `Spinner` for loading states

### Refactoring
- Completely redesigned **component structure**
- Migration to **Zustand** for state management
- Consistent **type imports** across codebase
- Improved **ESLint configuration**

## Technical Changes

- **better-auth** for authentication
- **Neon PostgreSQL** database integration
- **Kysely** as type-safe query builder
- **next-intl** for internationalization
- **MDX support** with Turbopack

## Breaking Changes

- Routes changed from `/[lang]/` to `/[locale]/`
- Old components from `src/components/UI/` removed
- New component imports from `@/components/ui/`
